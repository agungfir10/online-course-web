Langkah Awal Menjadi Web Developer

HTML

Struktur untuk membuat suatu website, merupakan bahasa markup.

CSS

Digunakan untuk mempercantik dan mendesain layout agar responsive digunakan di perangkat lain.

Javascript

Bahasa yang digunakan untuk membuat website lebih dinamis, atraktif, dan lebih berkomunikasi dengan pengguna.

PHP

Bahasa pemrograman yang digunakan untuk mengelola suatu data, memproses dan menampilkan.
